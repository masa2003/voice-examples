# voice-examples

Простой пример управления компьютером голосом

Для демонстрации используется управление плеером `audacious`, громкостью `pipewire`, и поиск через дефолтный браузер. Можно попросить "Компьютер, включи музыку", "Компьютер, следующий трек", "Компьютер, расскажи в каком году была октябрьская революция", "Компьютер уменьши звук на 50%" и т.д

### Зависимости 
```
linux, python, make, ffmpeg, pipewire
```

### Запуск
```
make
```

## Модели

По умолчанию используется vosk тулкит с моделью `vosk-model-small-ru-0.22` https://alphacephei.com/vosk/models. Поменять модель можно в `vosk_listener.py`

Так же можно запустить асистента с openai-whisper моделями:
```
make run-whisper
```
Осторожно, скачивается 5GB+ пакетов и модель на почти 2GB. Сами модели сохраняются в ~/.cache/vosk и ~/.cache/whisper 

Модель настраивается в `whisper_listener.py`. Для нормальной производетельности нужна CUDA

## Распознавание без ассистента

Можно просто потестировать распознавание в реальном времени:

```
make test-vosk
```

```
make test-whisper
```
